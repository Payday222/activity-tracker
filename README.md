# everything-tracker

to install this on Linux, download the .AppImage, make it executable and run it.
to install this on macOS, download the .app from releases, unpack it and put the .app into your Applications folder. After that you can launch it like any other app.
to install this on windows, downoad the .exe from releases, and run it as any other aplication on windows.
This project is still under developement, so if you have any suggestions about features that you think should be added, don't hesitate about posting them.
